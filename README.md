
This repository contains the data tables and Python scripts used to generate the isotopic inputs, Monte Carlo simulations, figures, and sensitivity analyses for the manuscript:

“Contrasting controls on δ18O–δ2H and δ18O–δ17O systematics in a balance-filled lake”
Custado et al. (2025)

All isotope values are reported relative to VSMOW. Small delta notation (δ) is reported in per mil (‰). 
Δ′17O values are reported in per meg unless otherwise stated.

FILE DESCRIPTIONS

1. bl_d17o_datasheet.csv
Primary master dataset containing all measured isotope data used in this study. This includes lake water, inflow components (inlet and creeks), groundwater and springs, and precipitation samples. This file is the sole source of measured isotope values used throughout all scripts.

2. monthly_clim_parameters.csv
Monthly climatological parameters extracted from the North American Regional Reanalysis (NARR) dataset. These data are used to calculate evaporation-flux-weighted precipitation isotopic compositions and to provide temperature inputs for mass balance calculations.

3. isotopic_inputs.csv
Summary table of mean isotopic values and associated uncertainties for lake water, inflow components, and precipitation. This file is generated by custado_et_al_2025_summary_samples.py and is used as input for Monte Carlo simulations and plotting scripts.

4. custado_et_al_2025_summary_samples.py
Processes the raw isotope dataset (bl_d17o_datasheet.csv) to calculate:

Monthly and annual amount-weighted precipitation isotope values
Evaporation-flux-weighted precipitation isotope values
Volume-weighted inflow isotope compositions
Mean lake isotope composition
Associated uncertainties (standard error of the mean)
Outputs isotopic_inputs.csv.

5. lake_balance_functions_R1.py
Function library implementing:

Equilibrium and kinetic isotope fractionation factors
Mass balance equations
Conversions between delta and ratio notation
Linearization and de-linearization of isotope values
Volume-weighted mixing calculations

This file is imported by all Monte Carlo, plotting, and sensitivity scripts.

6. custado_et_al_2025_mc_output_R1.py
Runs Monte Carlo simulations to estimate:

Relative humidity (h)
Evaporation-to-inflow ratio (Xe)
Atmospheric moisture isotopic composition
Input distributions are generated from measured isotope data and climatological parameters.

7. custado_et_al_2025_plots_R1.py
Generates main-text Figure 2

8. custado_et_al_2025_sensitivity_plot_R1.py
Performs sensitivity analyses by perturbing isotopic and climatic input parameters within interquartile ranges to evaluate response of h, Xe, and lake isotopic composition.

COLUMN DESCRIPTIONS — bl_d17o_datasheet.csv

Sample_ID
Unique identifier assigned to each water sample.

Sample_type
Category of sample. Possible values include:
Lake, Inlet, Creek, Ground, Spring, Precipitation.

Site_name
Name of sampling site or station (e.g., Bear River Inlet, Paris Creek, UVU precipitation station)
UVU: Utah Valley University
USU: Utah State University


Sample_Collection_Date
Date the sample was collected (YYYY-MM-DD).

d18O
Measured δ18O value (‰, VSMOW).

d18O_unc
Analytical uncertainty associated with δ18O (‰).

d17O
Measured δ17O value (‰, VSMOW).

d17O_unc
Analytical uncertainty associated with δ17O (‰).

D17O
Δ′17O (per meg), calculated using linearized delta notation and a reference slope (λ = 0.528).

D17O_unc
Analytical uncertainty associated with Δ′17O (per meg).

dD
Measured δ2H value (‰, VSMOW).

dD_unc
Analytical uncertainty associated with δ2H (‰).

d_exc
d-excess, calculated as δ2H − 8 × δ18O (‰).

d_exc_unc
Uncertainty associated with d-excess (‰), propagated from δ18O and δ2H uncertainties.

MeanElevation  
Mean elevation of the delineated watershed for each sampling site (meters above sea level), calculated from a digital elevation model (DEM) in ArcGIS.
